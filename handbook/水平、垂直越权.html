# е№іиЎҢи¶ҠжқғгҖҒеһӮзӣҙи¶Ҡжқғ

йҖ»иҫ‘жјҸжҙһ => йӘҢиҜҒз Ғз»•иҝҮгҖҒеҜҶз ҒжүҫеӣһгҖҒи¶ҠжқғгҖҒж”Ҝд»ҳжјҸжҙһ

йҖ»иҫ‘жјҸжҙһжңүдёҖдёӘеҘҪеӨ„пјҢдҪ еҸҜд»Ҙд»Җд№ҲйғҪдёҚжҮӮпјҢеҘ—е°ұе®ҢдәӢдәҶгҖӮдҪҶжҳҜжҖқз»ҙдёҖе®ҡиҰҒеҸ‘ж•Ј

ж°ҙе№іи¶Ҡжқғпјҡ зӨҫеҢәзҷ»еҪ•дәҶAз”ЁжҲ·иҙҰжҲ·гҖӮдҪҶжҳҜеҸҜд»ҘзңӢеҲ°Bз”ЁжҲ·зҡ„дҝЎжҒҜеҸӘдёҚиҝҮеҸӘжңүеҗҢзӯүжқғйҷҗгҖӮ


еһӮзӣҙи¶Ҡжқғпјҡ еҗ‘дёҠпјҡдҪҺжқғйҷҗз”ЁжҲ·еҸҜд»Ҙж“ҚдҪңй«ҳжқғйҷҗзҡ„дёңиҘҝ еҗ‘дёӢпјҡй«ҳжқғйҷҗз”ЁжҲ·еҸ—еҲ°йҷҗеҲ¶еҜјиҮҙе®һйҷ…еҸӘжңүдҪҺжқғйҷҗзҡ„жҺ§еҲ¶жқғ

иҝҳжңүдёҖз§Қзү№ж®Ҡжғ…еҶөдәӨеҸүи¶Ҡжқғе°ұжҳҜж—ўеҸҜд»Ҙж°ҙе№іпјҢд№ҹеҸҜд»ҘеһӮзӣҙ

# жңӘжҺҲжқғи®ҝй—®

иҝҳжңүдёҖз§Қзұ»еһӢжҳҜжңӘжҺҲжқғи®ҝй—®пјҢдёҘж јж„Ҹд№үдёҠиҖҢиЁҖиҝҷдёӘдёҚеұһдәҺи¶ҠжқғжјҸжҙһпјҢдҪҶжҳҜеңЁж—ҘеёёжөӢиҜ•дёӯеёёеёёдјҡйҒҮи§Ғ

еҸӘиҰҒиҫ“е…ҘжӯЈзЎ®зҡ„зҪ‘еқҖе°ұеҸҜд»ҘзӣҙжҺҘи®ҝй—®пјҢдҫӢеҰӮ/adminй»ҳи®Өе°ұжҳҜзҷ»еҪ•пјҢзҷ»еҪ•еҗҺи·іиҪ¬еҲ°user.phpпјҢз„¶еҗҺдҪ е°ұеҸҜд»ҘзӣҙжҺҘи®ҝй—®user.phpпјҢеҸ‘зҺ°дҪ зӣҙжҺҘжңүеҗҺеҸ°жқғйҷҗ

жҠҠжҸЎдҪҸдёҖдёӘдј еҸӮе°ұиғҪжҠҠжҸЎжё—йҖҸжөӢиҜ•зҡ„е‘Ҫи„үпјҒпјҒ

# и¶ҠжқғжөӢиҜ•

зҷ»еҪ•Aз”ЁжҲ·пјҢжӯЈеёёжӣҙж”№жҲ–жҳҜжҹҘзңӢAз”ЁжҲ·зҡ„дҝЎжҒҜпјҢз„¶еҗҺжҠ“еҸ–ж•°жҚ®еҢ…пјҢе°Ҷдј еҸӮIDдҝ®ж”№дёәе…¶д»–з”ЁжҲ·пјҢеҰӮжһңжҲҗеҠҹжҹҘзңӢеҲ°жҲ–иҖ…дҝ®ж”№дәҶеҗҢжқғйҷҗе…¶д»–з”ЁжҲ·зҡ„дҝЎжҒҜе°ұеұһдәҺж°ҙе№іи¶ҠжқғжөӢиҜ•гҖӮпјҲеҰӮжһңеҸҜд»ҘеҪұе“ҚеҲ°й«ҳжқғйҷҗз”ЁжҲ·е°ұжҳҜеһӮзӣҙи¶Ҡжқғпјү

дј еҸӮIDеҸӮж•°йңҖиҰҒиҮӘе·ұжЈҖжөӢпјҲеёёи§Ғпјҡuid= id= user= зӯүпјүйҖҡеёёдҪҝз”ЁburpиҝӣиЎҢзҲҶз ҙдј еҸӮпјҲдј еҸӮеҸҜиғҪеңЁGET POST COOKIEпјү

еёёи§Ғе№іиЎҢи¶ҠжқғпјҲдёҚйңҖиҰҒиҫ“е…ҘеҺҹеҜҶз Ғзҡ„дҝ®ж”№еҜҶз ҒпјҢжҠ“еҢ…ж”№з”ЁжҲ·еҗҚжҲ–иҖ…з”ЁжҲ·idдҝ®ж”№д»–дәәеҜҶз Ғпјү

дҝ®ж”№иө„ж–ҷзҡ„ж—¶еҖҷдҝ®ж”№з”ЁжҲ·id

жҹҘзңӢи®ўеҚ•зҡ„ж—¶еҖҷйҒҚеҺҶи®ўеҚ•id

# ж”Ҝд»ҳжјҸжҙһ

е•ҶжҲ·зҪ‘з«ҷжҺҘе…Ҙж”Ҝд»ҳз»“жһңжңүдёӨз§Қж–№ејҸпјҢдёҖз§ҚжҳҜйҖҡиҝҮжөҸи§ҲеҷЁиҝӣиЎҢи·іиҪ¬йҖҡзҹҘпјҢдёҖз§ҚжҳҜжңҚеҠЎеҷЁејӮз«ҜеҗҢжӯҘйҖҡзҹҘ

  • жөҸи§ҲеҷЁи·іиҪ¬

еҹәдәҺз”ЁжҲ·и®ҝй—®зҡ„жөҸи§ҲеҷЁпјҢеҰӮжһңз”ЁжҲ·еңЁй“¶иЎҢйЎөйқўж”Ҝд»ҳжҲҗеҠҹеҗҺпјҢзӣҙжҺҘе…ій—ӯдәҶйЎөйқўпјҢе№¶жңӘзӯүеҫ…й“¶иЎҢи·іиҪ¬еҲ°ж”Ҝд»ҳз»“жһңйЎөйқўпјҢйӮЈд№Ҳе•ҶжҲ·зҪ‘з«ҷе°ұж”¶дёҚеҲ°ж”Ҝд»ҳз»“жһңзҡ„йҖҡзҹҘпјҢеҜјиҮҙж”Ҝд»ҳз»“жһңйҡҫд»ҘеӨ„зҗҶгҖӮиҖҢдё”жөҸи§ҲеҷЁз«Ҝж•°жҚ®еҫҲе®№жҳ“иў«зҜЎж”№иҖҢйҷҚдҪҺе®үе…ЁжҖ§

  • жңҚеҠЎеҷЁз«ҜејӮжӯҘйҖҡзҹҘ

иҜҘж–№ејҸжҳҜж”Ҝд»ҳе…¬еҸёжңҚеҠЎеҷЁеҗҺеҸ°зӣҙжҺҘеҗ‘з”ЁжҲ·жҢҮе®ҡзҡ„ејӮжӯҘйҖҡзҹҘurlеҸ‘йҖҒеҸӮж•°пјҢйҮҮз”ЁPOSTжҲ–GETж–№ејҸгҖӮе•ҶжҲ·зҪ‘з«ҷжҺҘеҸ—ејӮжӯҘеҸӮж•°зҡ„urlеҜ№еә”зҡ„зЁӢеәҸдёӯпјҢиҰҒж”Ҝд»ҳе…¬еҸёиҝ”еӣһзҡ„ж”Ҝд»ҳз»“жһңиҝӣиЎҢзӯҫеҗҚйӘҢиҜҒпјҢжҲҗеҠҹеҗҺиҝӣиЎҢж”Ҝд»ҳйҖ»иҫ‘еӨ„зҗҶпјҢеҰӮйӘҢиҜҒйҮ‘йўқгҖҒи®ўеҚ•дҝЎжҒҜжҳҜеҗҰеҸ‘иө·ж”Ҝд»ҳж—¶дёҖиҮҙпјҢйӘҢиҜҒжӯЈеёёеҲҷеҜ№и®ўеҚ•иҝӣиЎҢзҠ¶жҖҒеӨ„зҗҶжҲ–дёәз”ЁжҲ·иҝӣиЎҢзҪ‘з«ҷеҶ…е…ҘиҙҰзӯү

  • и¶Ҡжқғж”Ҝд»ҳ

еӯҳеңЁusre=idиҝҷз§Қдј еҸӮзҡ„ж—¶еҖҷпјҢе°қиҜ•ж”№еҸҳidпјҢе°ұжңүеҸҜиғҪе°қиҜ•з”ЁеҲ«дәәзҡ„й’ұеҢ…д№°иҮӘе·ұзҡ„еҢ…еҢ…
